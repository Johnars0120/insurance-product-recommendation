保险产品智能推荐系统

面向课程实践的轻量级运营控制台，当前支持数据概览、逻辑回归训练、模型评估和推荐预测 API。

数据概览 查看训练集与评估集统计 样本数、字段数、目标列和正负样本比例。 模型训练 触发逻辑回归训练 API 页面只展示入口，训练由 POST 接口执行。 模型评估 查看最近一次运行指标 Accuracy、Precision、Recall、F1 和 AUC。 推荐预测 生成客户推荐等级 通过 API 返回概率、等级和推荐理由。
