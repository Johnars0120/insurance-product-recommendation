本文档用于最终交付前的浏览器演示。完整答辩讲解可参考 [答辩演示流程](答辩演示流程.md)。

## 预检测试与健康检查

```powershell
python -m pytest -q
```

最终验证结果为 `78 passed`。

## 启动服务

```powershell
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

启动后访问：

```text
http://127.0.0.1:8000/health
```

正常响应：

```json
{"status":"ok","project":"insurance-product-recommendation"}
```

## 演示顺序

1. 打开 `/`，说明系统目标和完整页面闭环。
2. 打开 `/data`，展示训练集 5822 条、评估集 4000 条、字段 86 个，以及正负样本比例图。
3. 打开 `/train`，选择三种模型之一并点击训练。
4. 打开 `/evaluate`，展示 Accuracy、Precision、Recall、F1、AUC 和模型对比图。
5. 打开 `/recommend`，输入推荐数量并生成推荐结果。
6. 展示推荐等级、预测概率、推荐理由和推荐等级分布图。
7. 打开推荐历史并导出 CSV。
8. 打开 `/docs`，展示数据、模型、推荐和图表 API。
9. 展示 `docs/screenshots/pytest 测试通过截图.png`，说明最终自动化测试结果。
