# Phase 1 MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Turn the first-stage startup guide into a minimum runnable FastAPI system that can profile the Excel data, train one baseline model, expose evaluation metrics, and generate recommendation levels.

**Architecture:** Keep the FastAPI app thin: routers handle HTTP, services own data/model logic, templates render simple first-stage pages. Generated models and exports remain ignored by git; tests exercise service behavior and API contracts with the real bundled sample data.

**Tech Stack:** FastAPI, Jinja2, pandas, scikit-learn, joblib, pytest, TestClient.

---

        assert 0.0 <= result["metrics"][metric] <= 1.0

    client = TestClient(app)

    response = client.post("/api/models/train", json={"model_name": "logistic_regression"})

    assert response.status_code == 200
    assert response.json()["metrics"]["auc"] >= 0.0


def test_evaluate_api_returns_latest_metrics_after_training():
    client = TestClient(app)
    client.post("/api/models/train", json={"model_name": "logistic_regression"})

    response = client.get("/api/models/evaluate")

    assert response.status_code == 200
    assert response.json()["model_name"] == "logistic_regression"
    assert "f1" in response.json()["metrics"]
