## 答辩流程模板

`docs/答辩演示流程.md` 建议包含：

```markdown
# 答辩演示流程

## 演示顺序

1. 启动系统。
2. 打开首页，介绍系统目标。
3. 打开数据页，展示数据统计和正负样本比例。
4. 打开训练页，训练逻辑回归、决策树、随机森林。
5. 打开评估页，展示模型指标对比。
6. 打开推荐页，生成推荐结果。
7. 导出推荐结果 CSV。
8. 打开接口文档 `/docs`。
9. 展示测试记录和成员分工。

## 讲解重点

- 数据存在类别不平衡，不能只看 Accuracy。
- 第二阶段增加 SQLite 持久化，训练和推荐记录可以保留。
- 多模型对比能帮助选择更合适的推荐模型。
- 图表和导出功能提升了系统展示和业务使用价值。
```
