# 推荐结果开发文档

## 负责人

成员 C：推荐结果模块。

## 工作目标

把推荐预测结果保存到 SQLite，支持查询最近推荐历史，并支持导出 CSV。答辩时可以演示“生成推荐结果 -> 查看历史 -> 导出文件”。


## 接口目标

| 接口 | 方法 | 目标 |
| --- | --- | --- |
| `/api/recommend/predict` | POST | 生成推荐结果，并保存到 SQLite |
| `/api/recommend/history` | GET | 查询最近一次或指定 `run_id` 的推荐结果 |
| `/api/recommend/export` | GET | 导出最近一次或指定 `run_id` 的推荐结果 CSV |

## 推荐结果字段

```text
customer_id
probability
recommend_level
reason
```

推荐等级规则保持第一阶段约定：

```text
high：probability >= 0.70
medium：0.40 <= probability < 0.70
low：probability < 0.40
```

## 开发步骤

1. 在 `history_service.py` 中增加保存推荐结果函数。
2. 在 `history_service.py` 中增加按 `run_id` 查询推荐结果函数。
3. 在 `history_service.py` 中增加查询最近推荐批次 `run_id` 的函数。
4. 修改 `predict_recommendations()`，预测成功后写入 `recommendation_results`。
5. 在 `recommendations.py` 中增加 `/history` 接口。
6. 在 `recommendations.py` 中增加 `/export` 接口，使用 `text/csv` 返回。
7. 测试导出内容包含 CSV header。
